# WildfireSim
Senior Project that simulates the possibilities of a wild fire spreading

# Background Info:
- 3 factors that primarily affect a wildfire
  - Topography
  - Weather 
  - Fuel
  - (https://www.nps.gov/articles/wildland-fire-behavior.htm)
